# AI-doptMe Mobile App

AI-doptMe is a mobile application that:
* Allows users to
  * Take or upload a photo of a dog
  * Enter a zip code
* Uses a custom trained Pet model to predict the dogs breed and color
* Finds a similar dog currently up for adoption at a nearby shelter

## Built With
* [Clarifai API](https://developer.clarifai.com/guide/)
* [React Native](https://facebook.github.io/react-native/docs/getting-started.html)
*  [Python 3](https://docs.python.org/3/library/index.html)
* [Flask](http://flask.pocoo.org/docs/0.12/)
